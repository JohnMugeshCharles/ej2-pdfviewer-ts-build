This sample showcases the annotation features of the PDF Viewer component, allowing users to add, modify, and manage annotations seamlessly within the document. Users can also customize annotations and interact with them directly through the viewer interface.

Explore the comprehensive annotation capabilities of the PDF Viewer component through this sample. Users can seamlessly work with annotations using the following functionalities:

  • Add and interact with different types of annotations directly on the PDF document.
  • Customize annotation properties such as color, opacity, and thickness based on user preference.
  • Manage and review annotations efficiently using the built-in comment panel.
  • Draw freehand annotations using the Ink tool for a natural writing experience.
  • Use the Ink Eraser, which is dynamically enabled when the Ink tool is selected, to remove ink strokes interactively. (New)
  • Filter annotations through the comment panel to quickly view and manage specific annotation types. (New)

More information on adding annotation programmatically can be found in this documentation section.
